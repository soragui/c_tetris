# 第 9 课：调试技巧 🐛

## 9.1 编译警告

```bash
CFLAGS = -Wall -Wextra -std=c99
```

## 9.2 printf 调试

```c
printf("DEBUG: 方块位置 (%d, %d)\n", 
       game->current.x, game->current.y);
```

## 9.3 常见问题

### 方块显示乱码
**解决：** 使用 `ACS_CKBOARD` 代替中文字符

### 方块下落太快
**解决：** 调整 `drop_interval = 3000`（3 秒）

### 碰撞检测失效
**调试：** 打印坐标检查计算是否正确

## 9.4 内存检查

```bash
valgrind --leak-check=full ./tetris
```
