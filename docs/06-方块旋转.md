# 第 6 课：方块旋转 🔄

## 6.1 旋转原理

4x4 矩阵顺时针旋转 90 度：

```
新位置 [j][3-i] = 原位置 [i][j]
```

## 6.2 代码实现

```c
void tetromino_rotate(Tetromino* t) {
    int temp[4][4];
    
    // 顺时针旋转
    for (int i = 0; i < 4; i++) {
        for (int j = 0; j < 4; j++) {
            temp[j][3-i] = t->shape[i][j];
        }
    }
    
    memcpy(t->shape, temp, sizeof(t->shape));
    t->rotation = (t->rotation + 1) % 4;
}
```

## 6.3 墙踢（Wall Kick）

旋转碰撞时，尝试左右移动一格来"踢"开墙壁。
