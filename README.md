# 🧱 Tetris - 俄罗斯方块

经典的俄罗斯方块游戏，使用纯 C 语言和 ncurses 库实现。

## 游戏截图

```

│      ██            │
│                    │  SCORE:   LEVEL:  LINES:

```

## 功能特性

- ✅ 7 种标准方块 (I, O, T, S, Z, J, L)
- ✅ 方块旋转和墙踢
- ✅ 下一个方块预览
- ✅ 方块保留 (Hold) 功能
- ✅ 软降和硬降
- ✅ 消行检测和计分
- ✅ 等级系统（速度递增）
- ✅ 暂停/继续
- ✅ 游戏结束检测

## 控制方式

| 按键 | 功能 |
|------|------|
| ←/→ 或 H/L | 左右移动 |
| ↑ 或 K/W | 旋转 |
| ↓ 或 J | 软降 |
| Space | 硬降 |
| C | 保留方块 |
| P | 暂停 |
| Q | 退出 |
| R | 重新开始 (游戏结束时) |

## 编译和运行

### 依赖

```bash
# Ubuntu/Debian
sudo apt install libncurses5-dev

# Fedora/RHEL
sudo dnf install ncurses-devel
```

### 编译

```bash
make          # 编译
make run      # 编译并运行
make clean    # 清理
```

## 计分规则

| 消行数 | 基础分 | 实际分 (等级 n) |
|--------|--------|----------------|
| 1 行   | 40     | 40 × (n+1)     |
| 2 行   | 100    | 100 × (n+1)    |
| 3 行   | 300    | 300 × (n+1)    |
| 4 行   | 1200   | 1200 × (n+1)   |

软降：每格 1 分  
硬降：每格 2 分
