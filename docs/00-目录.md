# 🧱 俄罗斯方块 - C 语言学习教程

这是一个为零基础 C 语言初学者设计的项目教程。通过动手实现经典的俄罗斯方块游戏，你将学会 C 语言的核心概念和 Linux 开发环境的使用。

## 🎯 学习目标

完成本教程后，你将能够：

- ✅ 理解 C 语言的基本语法和编程概念
- ✅ 掌握二维数组和结构体的使用
- ✅ 学会使用 ncurses 库进行终端编程
- ✅ 理解游戏循环和状态机的概念
- ✅ 能够独立设计和实现小型 C 语言项目

## 📋 前置要求

- 一台运行 Linux 的电脑（或虚拟机/WSL）
- 基本的电脑操作能力
- 不需要任何编程经验！

## 💡 学习建议

1. **动手实践** - 不要只看，要亲手敲代码
2. **理解原理** - 每步都要问"为什么"
3. **循序渐进** - 不要跳章节，基础很重要
4. **善用调试** - 学会使用 printf 调试
5. **记录笔记** - 好的笔记是成功的一半
